What Is Call Center Setup Training?

Call Center Setup Training is School of Freelancing's 20-hour, instructor-led course that teaches Linux users to build a production open-source call center — FreeSWITCH/FusionPBX with SSL, Telnyx & Twilio SIP trunks, IVR menus, and call queues. It covers everything from SIP/RTP fundamentals to live carrier integration, monitoring, and CDR reporting, then shows how to apply those skills to win freelance VoIP/call center clients.

Who This Is For

Beginners to intermediate Linux users who want job-ready VoIP/call center deployment skills (FusionPBX/FreeSWITCH + Telnyx/Twilio SIP trunking) and the ability to freelance on Upwork, Freelancer.com, Guru.com, and via direct LinkedIn outreach. You should be able to commit to 20 hours of hands-on, instructor-led training over 2 weeks.

Curriculum: 10 Sessions × 2 Hours (20 Hours Total)

Week 1 — Core Call Center Infrastructure

Session 1: VoIP & Call Center Fundamentals (2h)

  • SIP, RTP, PBX concepts, call flow basics
  • Open-source stack overview: FreeSWITCH, FusionPBX
  • Server provisioning on Ubuntu (DigitalOcean VPS)
🧪 Lab: Spin up a fresh Ubuntu VPS for the PBX server

Session 2: FreeSWITCH/FusionPBX Installation (2h)

  • Installing FreeSWITCH + FusionPBX
  • Web admin panel setup, initial configuration
  • Firewall (UFW) & port planning for SIP/RTP
🧪 Lab: Install and access FusionPBX admin dashboard

Session 3: Domain, SSL & Secure Access (2h)

  • Pointing a domain to the PBX server
  • SSL/HTTPS for the FusionPBX admin panel
  • Securing SSH and admin access
🧪 Lab: Attach domain, enable HTTPS, lock down server access

Session 4: Extensions & Internal Calling (2h)

  • Creating extensions, users, and ring groups
  • Softphone setup (Zoiper/Linphone) and internal test calls
  • Voicemail, call recording basics
🧪 Lab: Configure 3+ extensions and complete internal test calls

Session 5: Telnyx SIP Trunk Integration (2h)

  • Telnyx account setup, DID number purchase
  • Configuring Telnyx SIP trunk in FusionPBX
  • Inbound/outbound routing via Telnyx
🧪 Lab: Make and receive a live call through Telnyx

Week 2 — Advanced Setup & Freelance Readiness

Session 6: Twilio SIP Trunk Integration (2h)

  • Twilio account setup, Elastic SIP Trunking
  • Configuring Twilio trunk alongside Telnyx (multi-carrier setup)
  • Failover/least-cost routing between carriers
🧪 Lab: Make and receive a live call through Twilio

Session 7: IVR, Call Queues & Routing (2h)

  • Building IVR menus (press 1 for sales, etc.)
  • Call queues, agent groups, time conditions
  • Call routing logic for business hours/holidays
🧪 Lab: Build a working IVR with a call queue

Session 8: Monitoring, CDR & Maintenance (2h)

  • Call Detail Records (CDR) review and reporting
  • Monitoring active calls, server health, logs
  • Backups of PBX configuration and recordings
🧪 Lab: Configure CDR reporting + automated backup

Session 9: Client Project Simulation (2h)

  • Full end-to-end task: fresh VPS → FusionPBX → domain/SSL → Telnyx & Twilio trunks → IVR/queue
  • Troubleshooting common issues (one-way audio, NAT/RTP issues, registration failures)
  • Documentation & handover practices (client-ready reports)
🧪 Lab: Complete a timed "client ticket" simulation

Session 10: Freelance Client Acquisition (2h)

  • Building winning Upwork/Freelancer.com/Guru.com profiles & portfolios
  • Writing proposals that win VoIP/call center gigs
  • Pricing your services (hourly vs fixed-price gigs)
  • LinkedIn strategy: optimizing profile, outreach scripts, content posting for direct clients
🧪 Final Assessment: Submit 1 Upwork proposal + 1 LinkedIn outreach message (reviewed live)

Deliverables Upon Completion

  • A live portfolio project (your own SSL-secured FusionPBX server with Telnyx & Twilio trunks, IVR, and call queues)
  • Ready-to-use Upwork/Freelancer/Guru profile & proposal templates
  • LinkedIn outreach message templates for direct client acquisition
  • Access to School of Freelancing community for ongoing support

Tools & Platforms Covered

Ubuntu Linux, FreeSWITCH, FusionPBX, Telnyx, Twilio, SIP/RTP, UFW, Nginx, Certbot/Let's Encrypt, Zoiper/Linphone, Upwork, Freelancer.com, Guru.com, LinkedIn
